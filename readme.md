## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/wd301.git
    cd wd301
    ```
2. Install dependencies:
    ```bash
    npm install
    ```

## Usage

Start the development server:
```bash
npm start
```
Visit `http://localhost:3000` in your browser.
